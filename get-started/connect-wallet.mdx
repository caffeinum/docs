## Supported Wallets

Moai Chain works with any EVM-compatible wallet:

- MetaMask
- WalletConnect
- Coinbase Wallet
- Trust Wallet

## Add Network to MetaMask

1. Open MetaMask and click "Add Network"
2. Click "Add Network Manually"
3. Enter network details:

```bash
Network Name: Moai Chain Testnet
RPC URL: https://rpc.moai.ltd
Chain ID: 80418041
Currency Symbol: MOAI
Block Explorer: https://explorer.moai.ltd
```

## Add Using WalletConnect

1. Visit [app.moai.ltd](https://app.moai.ltd)
2. Click "Connect Wallet"
3. Select WalletConnect
4. Scan QR code with your wallet

## Programmatic Connection

Using ethers.js:

```typescript
import { ethers } from 'ethers';

const provider = new ethers.providers.JsonRpcProvider('https://rpc.moai.ltd');
const signer = provider.getSigner();

// Check connection
const network = await provider.getNetwork();
console.log('Connected to:', network.name);
```

Using web3.js:

```javascript
import Web3 from 'web3';

const web3 = new Web3('https://rpc.moai.ltd');

// Check connection
const chainId = await web3.eth.getChainId();
console.log('Connected to chain ID:', chainId);
```
