## Setup Development Environment

1. Install Node.js (v19 or higher):
```bash
# Using nvm (recommended)
nvm install 19
nvm use 19
```

2. Install Foundry:
```bash
curl -L https://foundry.paradigm.xyz | bash
foundryup
```

## Network Configuration

Add Moai Chain to your development environment:

```bash
# Chain Details
Chain ID: 80418041
RPC URL: https://rpc.moai.ltd
Currency Symbol: MOAI
Explorer: https://explorer.moai.ltd
```

## Get Test Tokens

You can get testnet tokens in two ways:

1. Bridge from Optimism testnet at [bridge.moai.ltd](https://bridge.moai.ltd)
2. Use the faucet at [faucet.moai.ltd](https://faucet.moai.ltd)
